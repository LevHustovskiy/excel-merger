# Объединение Excel-отчётов со сводкой

Скрипт решает типовую офисную задачу: в папке лежат однотипные Excel-файлы
(отчёты по продажам за разные месяцы / от разных менеджеров), нужно склеить их
в одну таблицу и автоматически построить сводку.

## Что получается на выходе

Один файл `итог.xlsx` с двумя листами:

- **«Все данные»** — все строки из всех файлов + колонка «Файл-источник»;
- **«Сводка»** — продажи по каждому менеджеру (количество и сумма),
  отсортированные по убыванию, со строкой ИТОГО.

```
    Менеджер  Количество    Сумма
0  Кузнецова          81  1876800
1     Иванов          66  1335400
2    Сидоров          71  1083600
3    Петрова          58   789400
4      ИТОГО         276  5085200
```

## Технологии

- Python 3.10+
- `pandas` — чтение, объединение, группировка данных
- `openpyxl` — запись Excel и авто-подгонка ширины колонок

## Запуск

```bash
pip install -r requirements.txt

# 1. Сгенерировать тестовые отчёты (если нет своих данных)
python make_demo_data.py

# 2. Объединить
python merge_reports.py --input ./reports --output итог.xlsx
```

## Что демонстрирует проект

- пакетная обработка файлов (glob по папке, пропуск битых файлов с предупреждением);
- валидация структуры данных (проверка обязательных колонок);
- группировка и агрегация через pandas;
- генерация аккуратного Excel: несколько листов, авто-ширина колонок;
- CLI-интерфейс с аргументами (`argparse`).
